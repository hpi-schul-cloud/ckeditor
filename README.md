# CKEditor

Custom version of CKEditor5 for the Schulcloud-Verbund-Software-Client

## Install

```sh
npm i @hpi-schul-cloud/ckeditor
```

## Supported editor modes

- classic toolbar
- balloon toolbar

## Built-in plugins

- AutoFormat
- BlockQuote
- Bold
- Code
- Essentials
- Heading
- Highlight
- HorizontalLine
- Image
- ImageInsertViaUrl
- Italic
- Link
- List
- Math
- Paragraph
- RemoveFormat
- SpecialCharacters
- Strikethrough
- Subscript
- Superscript
- Table
- TableToolbar
- WordCount

Note: the included image plugins only support an image insertion via URL, not via upload

## Supported languages

- de
- en
- es
- uk
