# CKEditor

Custom version of CKEditor5 for the Schulcloud-Verbund-Software-Client

## Install

```sh
npm i @hpi-schul-cloud/ckeditor
```

## Built-in plugins

- BlockQuote
- Bold
- Code
- Essentials
- Heading
- Italic
- Link
- List
- Paragraph
- Subscript
- Superscript
- Underline

## Supported languages

- de
- en
- es
