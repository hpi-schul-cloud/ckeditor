# CKEditor

Custom version of CKEditor5 for the Schulcloud-Verbund-Software-Client

## Install

```sh
npm i @hpi-schul-cloud/ckeditor
```

## Built-in plugins

- BlockQuote
- Bold
- Code
- Essentials
- Heading
- Highlight
- HorizontalLine
- Italic
- Link
- List
- Paragraph
- RemoveFormat
- SpecialCharacters
- Strikethrough
- Subscript
- Superscript
- Table
- TableToolbar
- Underline

## Supported languages

- de
- en
- es
